# Middletown Self Storage Email

These are table-based email signatures for Middletown Self Storage.

## Contributing

Just clone the repository to your machine. Please make a branch before committing any changes, and please submit a pull request with your changes for review.

## Adding a new signature

Make a copy of the template in the filled-signatures folder, with a filename formatted like `MSS-BrandingUpdate-EmailSig-FirstnameLastname.html`.

All fields to be replaced in the template are denoted with `$` - e.g. `$FIRST_NAME`. Find these and replace them.
